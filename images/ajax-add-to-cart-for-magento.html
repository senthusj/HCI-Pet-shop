Overview

Add and remove products to the shopping cart directly from the products listing and the product details pages without reloading the whole page. Enhance your customer experience and increase your store conversion rate with this Ajax Cart Extension

  • Uses jQuery and jQuery UI functions
  • Customizable theme, apply any of the themes from the jQuery UI theme page ( http://jqueryui.com/themeroller/ ) to replace the default dialog box css
  • Works with simple, configurable, grouped and downloadable products
  • Displays last items added to cart on the dialog box
  • Updates the sidebar cart box
  • Updates the cart top link
  • Displays product qty. form before adding to cart (optional in the backend)
  • Works with related products (product details page) and cross-sells (shopping cart page) .
  • Responsive version: set window width break point in extension settings.
  • Compatible with all the other Mango Extensions: Ajax Pagination, Price Slider, Color Swatches
  • Easy 10 minutes installation
